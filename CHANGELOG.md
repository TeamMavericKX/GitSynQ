# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-13

### Added
- Initial release of GitSynq.
- Interactive `init` command for setup.
- `push` command to sync local changes to air-gapped servers.
- `pull` command to sync remote changes back to local.
- `status` command to check sync state.
- `config` command to manage settings.
- Comprehensive documentation.
